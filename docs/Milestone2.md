# PROJECT-MILESTONE2-team2-ArbitraTech

## Vision and Scope

### Vision
**"打造一个直观、可靠的Web应用，帮助加密货币交易者洞察DEX与CEX间的非原子套利潜力。"**

### 技术愿景
1. **智能化分析**：通过先进的算法模型，实时捕捉DEX与CEX间的微观价格差异
2. **精准化识别**：建立多维度的套利识别标准，包括时间窗口、交易成本等因素

### 用户体验愿景
1. **零门槛操作**：即使对区块链技术了解有限的交易者也能轻松使用
2. **实时可视化**：通过直观的图表，将复杂的套利机会转化为易于理解的视觉信息
3. **决策支持**：不仅展示机会，还提供风险评估和利润预期，辅助用户做出更明智的交易决策

### 行业价值愿景
1. **市场效率提升**：通过识别和利用套利机会，促进DEX与CEX间的价格收敛
2. **风险管理工具**：为机构投资者提供专业的套利监控

### Scope
1. **交易对**：仅限USDT/ETH
2. **交易所**：仅限Uniswap V3（DEX）和Binance（CEX）
3. **数据时间窗口**：2025年9月1日至30日（历史数据分析）
4. **核心功能**：
    - 数据获取与处理管道
    - 双交易所价格可视化对比
    - 非原子套利机会识别算法
    - 预期利润计算与展示
